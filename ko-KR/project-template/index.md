# 2 단계: 스크래치 블록을 더하는 법 { .activity }

이 단계는 어떻게 스크래치 코드를 더하는지 알려줍니다

## 단계별 체크리스트 { .check }

+ 스크래치 코드는 이렇게 더할 수 있습니다:

```blocks

? 클릭했을 때
(120 v) 도 방향 보기
펜 색깔을 [#FF0000] (으)로 정하기
펜 내리기
무한 반복하기
   (1) 만큼 움직이기
   벽에 닿으면 튕기기
end
```

	다음 웹사이트에서 [test](http://scratchblocks.codeclub.org.uk) 스크래치 코드를 시험해볼 수 있습니다, 또한 다음 웹사이트에서 [guide](http://wiki.scratch.mit.edu/wiki/Block_Plugin/Syntax) 스크래치 블록을 프로젝트 노트에 더하는 가이드가 있습니다.

+ 또한 스크래치 블록에 맞춰 색-코드를 텍스트에 넣을 수 있습니다.

	![screenshot](blocks.png)

	다음과 같은 예가 있습니다:

	+ `동작` {.blockmotion} blocks, 예를 들어 `(10) 만큼 움직이기` {.blockmotion} 혹은 `벽에 닿으면 튕기기` {.blockmotion};
	+ `이벤트` {.blockevents} blocks, 예를 들어 `녹색 깃발을 클릭했을 때` {.blockevents} 혹은 `[message v]을(를) 받았을 때` {.blockevents};
	+ `형태` {.blocklooks} blocks, 예를 들어 `보이기` {.blocklooks} 혹은 `다음 모양으로 바꾸기` {.blocklooks}
	+ `제어` {.blockcontrol} blocks, 예를 들어 `무한 반복하기` {.blockcontrol}, 혹은 `(1) 초 기다리기` {.blockcontrol};
	+ `소리` {.blocksound} blocks, 예를 들어 `(야옹) 재생하기` {.blocksound} 혹은 `모든 소리 끄기` {.blocksound};
	+ `관찰` {.blocksensing} blocks, 예를 들어 `(스페이스)키를 눌렀는가?` {.blocksensing} 혹은 `... 묻고 기다리기` {.blocksensing};
	+ `펜` {.blockpen} blocks, 예를 들어 `펜 내리기` {.blockpen} 혹은 `지우기` {.blockpen};
	+ `연산` {.blockoperators} blocks, 예를 들어 `1 부터 10사이의 난수` {.blockoperators} 혹은 `10 * 2` {.blockoperators};
	+ 변수와 리스트엔 `데이터` {.blockdata} 블록, 예를 들어 `score` {.blockdata} 혹은 `(score)을(를) 1만큼 바꾸기` {.blockdata};
	+ `추가 블록` {.blockmoreblocks}, 예를 들어 `my function` {.blockmoreblocks}.
