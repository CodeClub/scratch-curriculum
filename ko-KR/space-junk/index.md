# 1 단계: 고양이 조종하기 { .activity }

플레이어가 방향키로 고양이를 조종할 수 있게 합시다.

## 단계별 체크리스트 { .check }

+ 새 스크래치 프로젝트를 시작하세요. 온라인 스크래치 에디터는 <a href="http://jumpto.cc/scratch-new">jumpto.cc/scratch-new</a> 에서 찾을 수 있습니다.

+ '축소' 버튼을 누르고 고양이를 몇번 눌러서 조금 작게 만들어주세요.

	![screenshot](sj-shrink.png)

+ 배경을 더하기 위해 '저장소에서 배경 선택'을 누르고 'Stars'를 선택해주세요.

	![screenshot](sj-bg.png)

	스테이지는 이렇게 보여야 합니다:

	![screenshot](sj-stage.png)

+ 왼쪽 방향키를 누르면 고양이가 왼쪽으로 가도록 만들어 보시다. 

	고양이 스프라이트를 누르고 다음 코드를 넣어주세요:

```scratch

[왼쪽 화살표 v] 키 눌릴 때
x좌표를 (-10) 만큼 바꾸기
```

	x 좌표를 바꿔야 고양이를 왼쪽으로 움직일 수 있다는데 주의해주세요.

	![screenshot](sj-coords.png)

+ 왼쪽 방향키를 눌러 고양이가 움직이는지 확인해보세요. 왼쪽으로 움직여야 할겁니다.

+ 위쪽 방향키를 눌렀을 때 위로 가게 하려면 이 코드를 추가해야 합니다:

```scratch

[위쪽 화살표 v] 키 눌릴 때
y좌표를 (10) 만큼 바꾸기
```

+ 프로젝트가 제대로 작동하는지 다시 확인해보세요!

# 2 단계: 우주 쓰레기! { .activity }

고양이가 피할 우주쓰레기를 만들어봅시다.

## 단계별 체크리스트 { .check }

+ '저장소에서 스프라이트 선택'을 누르시고 'Planet2'스프라이트를 스테이지에 넣어주세요.

	![screenshot](sj-planet.png)

+ 이 코드를 넣어 행성이 스테이지 주변을 반복해서 돌아다니도록 만드세요:

```scratch

? 클릭했을 때
무한 반복하기
   (2) 만큼 움직이기
   벽에 닿으면 튕기기
end
```

+ 초록색 깃발을 눌러서 행성이 제대로 작동하나 확인해보세요. 스테이지 왼쪽과 오른쪽을 왕복해야 할것입니다.

	행성이 화면 끝에 도착할때마다 뒤집어지는걸 보셨나요? 아래의 코드를 스크래치에 더해 문제를 해결하세요.

회전방식을 회전하지 않기 v 로 정하기


+ 고양이가 행성을 건드리면 게임이 끝나야 할것입니다. 이 코드를 행성의 `무한 반복하기` {.blockcontrol} 루프 __안쪽에__ 넣읍시다:

```scratch
만약 <[Cat1 v] 에 닿기?> 라면
   [모두 v] 멈추기
end
```

	지금까지 만든 행성 코드는 이렇게 되어 있어야 할 것입니다:

```scratch

깃발 클릭했을 때
무한 반복하기
   (2) 만큼 움직이기
   벽에 닿으면 튕기기
   만약 <[Cat1 v] 에 닿기?> 라면
      [모두 v] 멈추기
   end
end
```

+ 게임을 다시 시험해보세요 - 고양이가 행성을 건드리면 게임이 끝나나요?

# 3 단계: 지구로 돌아가기 { .activity }

우주 쓰레기를 30초간 피한 후 지구로 안전하게 돌아갈 수 있겠 나요?

## 단계별 체크리스트 { .check }

+ 일단 지구 스프라이트를 스테이지의 중심에 놓도록 합시다.

	![screenshot](sj-earth.png)

+ 이 코드를 지구 스프라이트에 넣어서 지구의 크기가 매우 작게 시작한 후 천천히 커져 보이게 만듭시다:

```scratch

깃발 클릭했을 때
크기를 (10)undefined로 정하기
무한 반복하기
   크기를 (0.3) 만큼 바꾸기
end
```

+ 지구 스프라이트를 확인해 보세요. 마치 고양이가 천천히 지구에 가까이 다가가는 듯이 보일 것입니다.

	![screenshot](sj-earth-big.png)

+ 스테이지를 클릭한 후 시간을 표기할 새 변수를 만드세요.

	![screenshot](sj-time.png)

	이제 스테이지 왼쪽 상단에 타이머가 보일 것입니다.

	![screenshot](sj-time-stage.png)

+ 이 코드를 스테이지에 더해서 타이머가 계속 세도록 만드세요:

```scratch

깃발 클릭했을 때
무한 반복하기
   (0.1) 초 기다리기
   [time v] 을(를) (0.1) 만큼 바꾸기
end
```

+ 초록 깃발을 눌러 시작해보면 타이머가 시간을 재기 시작하는 걸 볼 수 있을 것입니다.

	![screenshot](sj-time-test.png)

+ 게임의 승리조건은 고양이가 우주 쓰레기를 30초간 피하는 것입니다.

	이게 작동하려면 타이머를 처음엔 0초에 둔 후 타이머가 30초까지 올라갈 때까지 기다리면 됩니다.

	고양이 스프라이트를 클릭하고 이 스크립트를 더해주세요:

```scratch

깃발 클릭했을 때
[time v] 에 [0] 저장하기
<(time) > [30]> 까지 기다리기
[I made it!] 을(를) (2) 초동안 말하기
[모두 v] 멈추기

```
